# ML Churn: Underfitting vs Overfitting (Visual)

Notebook introductorio para predecir churn (clasificación binaria) y comparar modelos de ML
(Decision Tree, Logistic Regression, KNN, Random Forest) usando métricas Train vs Test.

## Contenido
- Generación de dataset (make_classification)
- Entrenamiento y evaluación de modelos
- Visualización Train vs Test (overfitting/underfitting)
